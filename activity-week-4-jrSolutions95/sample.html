Welcome to INFO 153A - Front end web-architecture

Course information

This course is a survey of technologies that power the user interfaces of web applications on various devices today. This course will explore the core front-end languages and frameworks and the underlying technologies that enable web applications. This course aims to provide an overview of the technical issues surrounding user interfaces powered by the web today and provide a solid and comprehensive perspective of the web's constantly evolving landscape. This course aims to demystify how the internet powers the user experience and demonstrates how you can understand and build it yourself.


This course is offered as INFO 153A for undergraduate students and INFO 253A for graduate students.

Both INFO 153A and 253A are held in the same lecture room; INFO 153A will require students to take a take-home practical, while INFO 253A will require students to complete a group-based final project and report.

In Fall 2020, INFO 153A did not exist: there was a single INFO 253A class that graduates and undergraduates with permission could take. This COCI submission proposes to make INFO 253A into an INFO 153A/253A class, with the changes documented for the undergraduate INFO 253A class mentioned below:

  • INFO 153A will have nine weekly lab assignments due throughout the semester. INFO 253A will have six weekly lab assignments and a final project. Note: The last three labs will be optional for INFO 253A students to attend and will cover some complex React concepts.
  • INFO 153A will have an open book Final Practical Exam, consisting of a project to complete in seven days during Finals Week. INFO 153A students will not participate in the final project.

If you want to learn about back-end systems, please check out Info 253B: Back-End Web Architecture, typically offered in the spring semester.
